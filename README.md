# Kronos Skin Switcher

Kronos Skin Switcher is a program add-on for **Kodi 21.x (Omega)** designed for **LibreELEC** and **Windows** systems. 
It automatically adjusts the Kodi skin color scheme based on the time of day, keeping the interface comfortable and visually consistent.

---

## Features
- **Time-Based Color Switching** – Changes skin colors dynamically:
  - Light in the morning and afternoon 
  - Default in the evening 
  - Darkest from late night 
- **Silent Operation** – Runs fully in the background without notifications or popups. 
- **Abort-Safe Startup** – Activates only after Kodi UI is ready. 
- **Arctic Zephyr Reloaded Support** – ONLY compatible with the popular Arctic Zephyr skin. 

---

## Compatibility
- Kodi **21 Omega** 
- LibreELEC 12.0.2+ 
- Windows 10/11 
- Arctic Zephyr Reloaded skin 
- Python 3 (bundled with Kodi)
